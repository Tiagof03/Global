# Detector de Mutantes ADN

Este repositorio contiene una solución para detectar, mutar y sanar secuencias de ADN en una matriz de 6x6.

## Integrantes del Grupo
- Tiago Funes 

## Estructura del Repositorio
- `clases.py`: Contiene las clases `Detector`, `Mutador`, `Radiacion`, `Virus`, y `Sanador`.
- `ejecutable.py`: Programa ejecutable que interactúa con el usuario.
- `README.md`: Instrucciones y descripción del proyecto.

## Cómo Ejecutar el Programa

1. Clona este repositorio en tu máquina local.
2. Asegúrate de tener Python instalado.
3. Ejecuta el archivo `ejecutable.py` desde la terminal:
   ```bash
   python ejecutable.py

Ejemplo de ejecución:

Seleccione una opción:
1: Detectar mutaciones
2: Mutar ADN
3: Sanar ADN
4: Salir
Opción: 1
El ADN contiene mutaciones.



Este README cumple con las especificaciones, proporcionando una descripción clara y cómo ejecutar el programa.
